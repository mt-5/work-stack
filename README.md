# Work Stack

All-in-one productivity tool for software developers

## [work-stack.vercel.app](https://work-stack.vercel.app)

### Setup on local machine

1. Install dependencies & run local server

   ```bash
   # npm
   npm install
   npm run dev
   mv .env.example .env
   ```

2. Update values in `.env` file
